# 📩 SMS Forwarder App

**Android Application – Developed by Shaik Adnan**

A lightweight and efficient Android application that automatically forwards incoming SMS messages to a specified number. Built with **Java** and **Android Studio**, this app demonstrates event-driven programming, broadcast receivers, and UI/UX design for real-world utility.

---

## ✨ Key Features
- 📲 **Automatic SMS Forwarding**: Forwards incoming SMS to a predefined number instantly.
- 🔔 **Real-Time Notifications**: Displays a notification when a message is forwarded.
- 🛡️ **Secure Handling**: Ensures message integrity during forwarding.
- 🗂️ **Message Log**: Keeps track of forwarded messages in a simple list view.
- ⚡ **Lightweight Design**: Optimized for performance and minimal resource usage.

---

## 🎨 UI/UX Highlights
- **Minimalist Layout**: Clean interface with intuitive navigation.
- **RecyclerView Integration**: Displays forwarded messages in a structured list.
- **Material Design Components**: Buttons, cards, and icons for a modern look.

---

## 🛠️ Tech Stack
- **Language**: Java  
- **IDE**: Android Studio  
- **Core Components**: BroadcastReceiver, RecyclerView, Adapter, XML Layouts  
- **Gradle**: Kotlin DSL build scripts
